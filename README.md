# 🤖 Livo.ai – LLM-Based Task Automation & Recommendation System

## 📌 Overview
Livo.ai is a GenAI-powered backend system that interprets natural language input and provides intelligent task recommendations using Large Language Models (LLMs).

## 🚀 Features
- Natural language query processing using LLM APIs  
- Intelligent task recommendation system  
- Backend automation workflows  
- SQL-based ranking and filtering logic  
- Scalable REST API design  
- Cloud deployment using AWS  

## 🛠️ Tech Stack
- Python  
- OpenAI API  
- Machine Learning  
- SQL  
- AWS (EC2, S3)
